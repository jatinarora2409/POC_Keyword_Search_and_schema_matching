New Build in 2020, The Harper Homes are a Modern, Gated & Secured Re-Developed Fourplex community, that consists of 4 separate single-family homes, with separate keyless entrances, and 2 patio spaces in each unit. Each house is strategically built with an open floorplan maximizing space, and the feel of new custom homes. Located in the best part of West Hollywood, each house shattered all lease records (price per SQFT), in the zip code. This building includes, but is not limited to; Gated Community, Security Cameras, Private Entrances into Each Unit, multiple patio areas in each unit, A new set of ALL Appliances in all units (Washer, Dryer, Refrigerator, Stove, Microwave, Dishwashers), High-End Granite and Tile inside and outside of each unit, Landscaping, custom closets, Vinyl waterproof flooring, light dimmers, etc. Nothing is leased, everything is owned.
